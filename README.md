# 🌿 Plant Disease Prediction Using Deep Learning (CNN)

An end-to-end **Streamlit web application** that classifies plant leaf diseases using a trained Convolutional Neural Network (CNN) model. Users can upload an image of a plant leaf and get instant disease predictions.

## 📌 Project Overview

This project uses a deep learning model trained on plant disease images to predict the type of disease from a given leaf image. The model is integrated into a simple and interactive Streamlit web app.

---

## 🧠 Model Details

- Architecture: CNN (Convolutional Neural Network)
- Framework: TensorFlow / Keras
- Input Size: 224 × 224 pixels
- Output: Multi-class plant disease classification
- Activation: Softmax
